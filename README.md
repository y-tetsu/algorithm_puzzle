## パズル
### 43. Number Placement
```
$ python p043_number_placement.py
numbers = 2 5 1 0
boxes   = [] < [] > [] < []
result  = 0 < 5 > 1 < 2

numbers = 87 26 84 42 20 12 53 88 75 94
boxes   = [] > [] < [] > [] > [] < [] > [] < [] < [] < []
result  = 94 > 12 < 88 > 87 > 20 < 84 > 26 < 42 < 53 < 75
```
